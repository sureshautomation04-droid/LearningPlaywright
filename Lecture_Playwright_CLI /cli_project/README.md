# CLI Project - Playwright Command-Line Toolkit

## Overview
A complete project demonstrating every Playwright CLI feature with runnable demo scripts and test files.

## How to Run

### Run All Tests
npx playwright test

### Run Demo Scripts
bash scripts/01_run_basic.sh

### Run All Demos
bash scripts/10_run_all_demos.sh

## Project Structure
- config/ - Project configuration
- scripts/ - Shell scripts for each CLI feature demo
- codegen_output/ - Pre-recorded codegen output examples
- tests/ - 10 Playwright test files (7 pass, 3 intentional fail)
- reporters/ - Custom TTA HTML reporter
- reports/ - Generated test reports
- traces/ - Generated trace files
